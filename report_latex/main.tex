\begin{abstract}
This report compares three approaches to estimate cardiac metrics from ultrasound data: geometric approximation from 2-D views, Doppler-based cardiac output, and a modified Simpson method implemented in Python. The workflow estimates end-diastolic and end-systolic ventricular volumes and derives stroke volume, ejection fraction, and cardiac output. The study also documents the software implementation used to perform frame selection, scale calibration, manual measurements, and final metric computation.
\end{abstract}
